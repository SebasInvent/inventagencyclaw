# InventAgency - Modelo de Negocio

## Modelo: Development on Demand

No se cobra por horas — se cobra por **experiencia/entregable**. Inspirado en el modelo DENOM de Jaqueline De Luna.

## Tipos de Cliente

### 1. Cliente Startup
- **Necesidad**: "Necesito un MVP rápido y escalable"
- **Servicio**: MVP Sprint (4-8 semanas)
- **Funcionalidades**: Cotizar, Ver prototipo IA, Agendar Call, Mis Proyectos, Chat

### 2. Cliente Enterprise
- **Necesidad**: "Modernizar mis sistemas sin parar operaciones"
- **Servicio**: Digital Transformation Package
- **Funcionalidades**: Auditoría Tech, Roadmap, Reportes, SLA Dashboard

### 3. Cliente Agency (White Label)
- **Necesidad**: "Entregar proyectos tech sin equipo propio"
- **Servicio**: White Label Development
- **Funcionalidades**: Portal de Proyectos, Branding Personalizado, Pricing mayorista

### 4. Cliente Pilot (Early Access)
- **Funcionalidades**: Mis Proyectos, Enviar Feedback, Features beta

### 5. Profesional/Developer
- **Funcionalidades**: Toggle Pro/Cliente, Gestión proyectos, Time tracking, Kanban

## Paquetes de Experiencias

| Paquete | Precio | Duración | Target |
|---------|--------|----------|--------|
| Discovery Sprint | $500 USD | 1 semana | Clientes nuevos |
| MVP Launch | Desde $8,000 USD | 4-8 semanas | Startups |
| Scale Up | Desde $15,000 USD | 8-12 semanas | Startups con tracción |
| Enterprise Transform | Desde $50,000 USD | 3-12 meses | Empresas |

## Sistema de Cotización

### Tipos de Proyecto (9)
1. App Móvil — $10,000 base
2. App Web — $8,000 base
3. API / Backend — $6,000 base
4. Inteligencia Artificial — $15,000 base
5. Blockchain / Web3 — $20,000 base
6. Biometría / Seguridad — $18,000 base
7. Dashboard / Analytics — $7,000 base
8. E-commerce — $12,000 base
9. Sistema Empresarial — $25,000 base

### Multiplicadores
- **Básico** (5-10 features): 1x
- **Intermedio** (10-20 features): 1.8x
- **Avanzado** (20+ features): 2.5x

### Ajustes de Timeline
- **Express** (4 semanas): +30%
- **Estándar** (8 semanas): 0%
- **Flexible** (12+ semanas): -10%

## KPIs de Negocio
- Cotizaciones generadas por mes
- Tasa de conversión cotización → proyecto
- Revenue por vertical
- Ticket promedio por proyecto
- LTV del cliente
