## 🔄 FLUJO DE TRABAJO

### Cómo Usar Este Sistema

1. **Crear orden:** Agregar a este archivo
2. **Asignar:** Quién lo hace
3. **Priorizar:** CRÍTICA / ALTA / MEDIA / BAJA
4. **Ejecutar:** Trabajar en la tarea
5. **Actualizar:** Marcar checkboxes [x]
6. **Sincronizar:** Commit a GitHub
7. **Cerrar:** Cuando todo esté [x]

### Sincronización

```bash
# Después de actualizar órdenes:
git add docs/SISTEMA-DE-ORDENES.md
git commit -m "orden: actualización de tareas"
git push origin main

# Sincronizar a Obsidian:
rsync -av docs/ ~/Documents/Obsidian-Vault/InventAgency/
```
