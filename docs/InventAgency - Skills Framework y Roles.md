# InventAgency - Skills Framework y Roles

## Roles del Equipo

### Leadership & Estrategia
1. **CEO / Founder** (Sebastian) — Visión, Negocio, Relaciones
2. **CTO / Technical Director** — Tecnología, Arquitectura, Innovación
3. **Project Manager / Delivery Lead** — Entrega, Coordinación, Clientes

### Desarrollo Técnico
4. **Senior Full Stack Developer** — Backend, Frontend, Database
5. **Blockchain Developer** — Smart Contracts, Web3, DeFi
6. **AI/ML Engineer** — Modelos, APIs, Data Processing
7. **Mobile Developer** — React Native / Flutter, iOS/Android
8. **DevOps / Cloud Engineer** — Infraestructura, CI/CD, Monitoring
9. **QA / Automation Engineer** — Calidad, Testing, Automation

### Diseño & UX
10. **UI/UX Designer** — Interfaces, Usabilidad, Design Systems
11. **Product Designer** — Producto, Estrategia, Discovery

### Negocio & Crecimiento
12. **Business Analyst** — Requisitos, Procesos, Documentación
13. **Sales / Account Manager** — Ingresos, Relaciones, Crecimiento
14. **Marketing / Content Strategist** — Brand, Lead Gen, Contenido

## Niveles de Competencia

| Nivel | Descripción | Dominio |
|-------|-------------|---------|
| Novato | Aprendiendo, requiere supervisión | 0-40% |
| Intermedio | Puede trabajar independientemente | 40-70% |
| Avanzado | Referente, mentoriza otros | 70-90% |
| Experto | Referente técnico, innova | 90-100% |

## Evaluación Trimestral
1. **Autoevaluación** (20%)
2. **Evaluación de pares** (30%)
3. **Evaluación del líder** (40%)
4. **Feedback de clientes** (10%)

## Plan de Desarrollo Individual

- Recursos asignados (cursos, mentorship, proyectos)
- Timeline de 3-6 meses
