# InventAgency Web - Landing Rediseño

## Estado: 🔄 En Progreso

Rediseño completo de la landing page estilo Skiper UI. Minimalista, futurista, con efectos sutiles pero impactantes.

## Stack
- **Framework**: Next.js 16
- **Styling**: TailwindCSS v4
- **Animations**: Framer Motion
- **Scroll**: Lenis (smooth scrolling)
- **React Compiler**: babel-plugin-react-compiler

## Concepto Visual
- Minimalista y futurista
- Efectos sutiles pero impactantes
- Animaciones suaves y fluidas
- Grid system moderno (Bento Grid)
- Glassmorphism y blur effects
- Magnetic interactions
- Modo claro/oscuro 100% funcional

## Componentes Base

### Completados ✅
1. BentoCard — Cards con hover effects
2. MagneticButton — Botones con efecto magnético
3. AnimatedGrid — Grid animado de fondo
4. FloatingElements — Partículas flotantes
5. GlowCard — Cards con efecto glow

### Pendientes ⏳
6. SpotlightCard — Card con efecto spotlight
7. TextReveal — Texto con animación de reveal
8. ParallaxSection — Secciones con parallax
9. InteractiveGrid — Grid interactivo para portfolio

## Secciones del Rediseño

1. **Hero** — Typing/reveal title, magnetic CTAs, animated grid bg, floating particles
2. **Services** — Bento grid, hover effects, glassmorphism
3. **Tech Stack** — Logos con hover, tooltips animados
4. **Portfolio** — Grid dinámico, spotlight cards, modal detalles
5. **About** — Timeline interactiva, stats animados
6. **Methodology** — Steps secuenciales, parallax cards
7. **Testimonials** — Carousel moderno, glassmorphism
8. **Contact** — Form con validación visual, focus effects
9. **Footer** — Minimalista, links con hover
