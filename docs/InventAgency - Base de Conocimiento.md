## Resumen Ejecutivo

**Invent Agency** es una agencia de desarrollo de software premium que opera bajo el modelo "Development on Demand". Conecta clientes (startups, enterprise, agencies) con equipos de desarrollo especializados en tecnologías de punta.

### Diferenciadores Clave
1. **No se cobra por horas** — Se cobra por experiencia/entregable
2. **Prototipos con IA** antes de desarrollar
3. **Estimaciones inteligentes** con cálculo automático
4. **Paquetes por vertical** (FinTech, HealthTech, Blockchain, Biometría)
5. **Visual liquid metal** — Estética futurista y diferenciada

### Proyectos Activos
| Proyecto | Estado | Stack | Descripción |
|----------|--------|-------|-------------|
| Platform MVP | Completado | Next.js + Supabase | Plataforma de cotización y booking |
| Web Landing | En rediseño | Next.js + Tailwind v4 | Landing estilo Skiper UI |
| ThreeJS Experience | Completado | Three.js + Vanilla | Experiencia 3D inmersiva |
