# InventAgency ThreeJS - Experiencia 3D

## Estado: ✅ Completado

Experiencia web inmersiva con Three.js. Landing page con efectos 3D, loader animado, y switch de idioma ES/EN.

## Stack
- **3D Engine**: Three.js (vanilla)
- **Styling**: CSS custom
- **i18n**: JSON files (en.json, es.json)
- **Font**: Space Grotesk

## Características
- Loader animado con barra de progreso
- Experiencia 3D interactiva
- Switch de idioma ES/EN
- Frames generados (181 frames para video)
- Diseño futurista coherente con la marca

## Estructura
```
inventagency-threejs/
├── index.html          # Página principal
├── script.js           # Lógica Three.js
├── styles.css          # Estilos custom
├── i18n/
│   ├── en.json         # Inglés
│   └── es.json         # Español
└── frames/             # 181 frames para video
    ├── frame_0001.jpg
    └── ...
```
